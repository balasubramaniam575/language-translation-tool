# Language Translation Tool

A Python-based language translation tool that translates text into multiple languages using the Deep Translator library.

## Features
- Automatic source language detection
- Supports multiple languages
- Command-line version
- GUI version using Tkinter

## Technologies Used
- Python
- Deep Translator
- Tkinter
- Git & GitHub

## Installation

pip install deep-translator

## Run CLI Version

python translator.py

## Run GUI Version

python translator_gui.py
